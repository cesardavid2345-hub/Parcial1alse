
Este es un programa en C++ que recibe un conjunto de puntos en el plano y calcula cuál tiene la mayor magnitud respecto al origen
El programa:
solicita al usuario la cantidad de puntos
 pide las coordenadas de cada punto
calcula la magnitud
 muestra el punto con mayor magnitud
Compilación:
g++ programa.cpp -o programa
Ejecución:
./programa
